# SNS副業ダッシュボード

iPhone/iPadで見やすい、HTML・CSS・JavaScriptだけのシンプルな副業/SNS管理Webアプリです。

## できること

- 投稿管理（投稿日、媒体、投稿タイトル、本文、Canva画像リンク、投稿ステータス）
- 問い合わせ管理（問い合わせ日、内容、対応状況）
- 作業予定管理（作業予定日、作業内容）
- 売上管理（売上金額、原価、利益）
- 月別集計
- CSV出力
- スマホ/iPad向けレスポンシブ表示

## 使い方

`index.html` をブラウザで開くだけで利用できます。現在は個人情報や実データを含まないサンプルデータのみを表示します。

## プッシュ前の安全確認結果

2026-06-07に以下を確認しました。

- APIキー、パスワード、アクセストークン、実在する個人情報は含めていません。
- JavaScript内に `fetch`、`XMLHttpRequest`、`sendBeacon`、`WebSocket` などの外部送信用処理はありません。
- Canva画像リンクは外部サービスへ遷移しないサンプル文字列にしており、外部サービス連携は行っていません。
- `.github/workflows`、GitHub Actions、課金サービス、決済サービス、自動実行につながる設定ファイルは追加していません。
- 構成ファイルは `index.html`、`styles.css`、`app.js`、`README.md` のみで、HTML/CSS/JavaScriptだけで動きます。
- レスポンシブCSSで820px以下と520px以下の表示を調整し、iPhone/iPadでカードやタブが見やすい構成にしています。
