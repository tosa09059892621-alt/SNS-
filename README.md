# SNS副業ダッシュボード

iPhone/iPadで見やすい、HTML・CSS・JavaScriptだけで動く副業・SNS管理用のシンプルなWebアプリです。

## 目的

副業で行うSNS投稿、問い合わせ、作業予定、売上を1画面で確認し、月別に状況を把握できるようにします。

## 管理できる項目

- 投稿日
- 媒体（Instagram、Facebook、Googleビジネスプロフィール）
- 投稿タイトル
- 投稿本文
- Canva画像リンク（サンプル文字列のみ）
- 投稿ステータス
- 問い合わせ日
- 問い合わせ内容
- 作業予定日
- 作業内容
- 売上金額
- 原価
- 利益
- 対応状況
- 改善メモ

## 機能

- 投稿管理
- 問い合わせ管理
- 作業予定管理
- 売上管理
- 月別集計
- CSV出力
- スマホ/iPad向けレスポンシブ表示

## 構成

- `index.html`：画面構造
- `styles.css`：スマホ/iPad向けの見た目とレスポンシブ調整
- `app.js`：サンプルデータ、表示切り替え、月別集計、CSV出力

## 使い方

`index.html` をブラウザで開くだけで利用できます。サーバーやビルドツールは不要です。

## データと通信について

- 個人情報、実データ、APIキー、パスワードは含めていません。
- 表示データはすべてサンプルです。
- API連携や外部サービス通信は行いません。
- GitHub Actionsは使用していません。

## マージ後も維持する条件

- `index.html`、`styles.css`、`app.js` の3ファイルで画面と機能を提供します。
- HTML/CSS/JavaScriptだけで動き、外部サービスへの通信は行いません。
- GitHub Actionsやサーバーサイド処理は使用しません。
- サンプルデータのみを使用し、APIキー、パスワード、個人情報は保存しません。
