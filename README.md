# SNS副業ダッシュボード

iPhone/iPadで見やすい、HTML・CSS・JavaScriptだけのシンプルな副業/SNS管理Webアプリです。外部サービス連携は行わず、サンプルデータから始めてブラウザ内に保存できます。

## できること

- 投稿管理（投稿日、媒体、投稿タイトル、本文、Canva画像リンク、投稿ステータス）
- 問い合わせ管理（問い合わせ日、内容、対応状況）
- 作業予定管理（作業予定日、作業内容）
- 売上管理（売上金額、原価、利益）
- 原価入力と利益自動計算（売上金額 - 原価）
- 月別集計（売上、利益、未対応問い合わせ、作業予定）
- CSV出力
- ブラウザ内保存（`localStorage` に保存）
- スマホ/iPad向けレスポンシブ表示

## 使い方

1. `index.html` をブラウザで開きます。
2. 「新規登録」フォームに投稿、問い合わせ、作業予定、売上、原価、改善メモを入力します。
3. 原価を入力すると利益が自動計算されます。
4. 「ブラウザに保存」を押すと、このブラウザ内にだけ保存されます。
5. 月別フィルターやタブで、投稿・問い合わせ・作業予定・売上を確認できます。
6. 必要に応じて「CSV出力」からデータを書き出せます。
7. デモ状態に戻したい場合は「サンプルに戻す」を押します。

## ファイル構成

- `index.html`：画面構成、入力フォーム、集計カード、一覧テンプレート
- `styles.css`：iPhone/iPad向けレスポンシブデザイン
- `app.js`：サンプルデータ、ブラウザ内保存、利益計算、月別集計、CSV出力
- `README.md`：説明と安全確認結果

## プッシュ前の安全確認結果

2026-06-07に以下を確認しました。

- APIキー、パスワード、アクセストークン、実在する個人情報は含めていません。
- 使用データはSNS運用を想定したサンプルデータのみです。
- JavaScript内に `fetch`、`XMLHttpRequest`、`sendBeacon`、`WebSocket` などの外部送信用処理はありません。
- Canva画像リンクは外部サービスへ遷移しないサンプル文字列にしており、外部サービス連携は行っていません。
- 保存先はブラウザ内の `localStorage` のみで、入力内容をサーバーや外部サービスへ送信しません。
- `.github/workflows`、GitHub Actions、課金サービス、決済サービス、自動実行につながる設定ファイルは追加していません。
- 構成ファイルは `index.html`、`styles.css`、`app.js`、`README.md` のみで、HTML/CSS/JavaScriptだけで動きます。
- レスポンシブCSSで820px以下と520px以下の表示を調整し、iPhone/iPadでカード、入力フォーム、タブが見やすい構成にしています。
