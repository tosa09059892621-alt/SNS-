# SNS副業ダッシュボード

iPhone/iPadで見やすい、HTML・CSS・JavaScriptだけで動く副業・SNS管理用のシンプルなWebアプリです。

## 目的

副業で行うSNS投稿、問い合わせ、作業予定、売上を1画面で確認し、月別に状況を把握できるようにします。

## 機能

- 投稿管理
- 問い合わせ管理
- 作業予定管理
- 売上管理
- 月別集計
- CSV出力
- スマホ/iPad向けレスポンシブ表示

## 管理項目

投稿日、媒体、投稿タイトル、投稿本文、Canva画像リンク、投稿ステータス、問い合わせ日、問い合わせ内容、作業予定日、作業内容、売上金額、原価、利益、対応状況、改善メモをサンプルデータで管理します。

## 使い方

`index.html` をブラウザで開くだけで利用できます。サーバーやビルドツールは不要です。

## データについて

個人情報、実データ、APIキー、パスワードは含めていません。外部サービス通信やGitHub Actionsも使用していません。
